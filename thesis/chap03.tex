\chapter{Metrics}
\label{sec:metrics}

The metrics can be divided into three categories:

\begin{itemize}
    \item plain corpus based,
    \item annotated corpus based,
    \item morphology segmentation based.
\end{itemize}

The metrics based on the plain corpus are the most easy-to-access with respect to the data used. Annotated corpus provides many interesting metrics, but the annotation would add a whole new level of complexity and it is for the expected complexity of a bachelor's thesis omitted. The added value would be nevertheless large.

Since one of the objectives of this theses is the morphological segmentation. These metrics the most relevant in this case. Most of them are abstraction of metrics that are applicable for words.

The following sections are describing all used metrics. 

\section{Corpus based metrics}

\subsection{Type to token ratio (TTR)}

$$
\frac{\text{\# Word types}}{\text{\# Tokens}}
$$

This metric is suggested by \textcite{Coltekin2022}. In this title it is calculated as average over floating window of fixed size. In the case of source data being the whole corpus, this metric can be calculated simply from the frequency list. Higher TTR should suggest the variability of paradigms and higher word-form count in a lexeme. Compare two corpora with $10$ tokens:
\begin{itemize}
    \item only $5$ word types leads to TTR being $\frac{1}{2}$
    \item only $10$ word types leads to TTR being $1$
\end{itemize}

Higher TTR might suggest higher complexity.

\subsection{Word-type entropy}
\parencite{Coltekin2022}

$$
H=-\sum_{w \in V} p(w)\log p(w)
$$
Where
\begin{itemize}
    \item $V$ is a set of all word types,
    \item $p(w)$ is the probability of the word-type $$\frac{\text{\# token count of one word-type}}{\text{\# all tokens}}$$.
\end{itemize}

Higher entropy means that the frequencies are distributed more evenly across many word types. Lower entropy means that relatively small amount of word types account for a large proportion of all tokens. In this sense, entropy captures the uncertainty of the word distribution: if the next token is difficult to predict because many word types have comparable probabilities, the entropy is high, which might suggest more complexity.

\textbf{Perplexity} can be derived from entropy as
$$
\mathrm{PP} = 2^H
$$
which can be interpreted as the effective number of equally probable word types.

\subsection{Information in word structure}
\parencite{Coltekin2022}
$$
\frac{\text{Size of compressed text}}{\text{Size of compressed text with randomized words}}
$$
This measure captures the information encapsulated in the word structure. The question is how much is the text compressible if letters are randomized. Words are randomized so that they are the same size, same proportions of letters in general, but no patterns.

There is a variant to use the same randomization for the same word-types.

This metric was calculated in following way:
\begin{enumerate}
    \item It was assigned a unique random 16 bit long sequence to all unicode characters used in all datasets.
    \item For each language, a file with $100\ 000$ tokens was created. Words were generated by random with respect to the frequency list.
    \item All letters in the file were replaced with the assigned random sequence. The tokens were separated by 16 bit long sequence of zeros.
    \item The file was compressed.
\end{enumerate}

\subsection{Hapax legomenon}
\parencite{Milicka2022}

Proportion of words that occurring only once in the text. It can represent how many unique word-types the speaker needs. This measure is however applicable to smaller datasets. In large corpora, it measures only the amount of noise present in the dataset. It is more useful in the morpheme part.

\subsection{Zipf slope}

Zipf law was explained shown on Figure~\ref{fig:freq_rank_three}. This metric can proof, whether it is true that all languages are governed with the same principal, or some of them are declining faster that others.

\subsection{Other statistics}

The basic statistical measures are listed here:
\begin{itemize}
    \item Average word length (of types and tokens)
    \item Word length variance
    \item Word-type size after normalization. The truncation was specifically optimized for original word-type count be independent on the truncation rule. It might be interesting to see how many word-types remained in the dataset. 
\end{itemize}

\section{Metrics based on segmentation}

Let's first introduce some more morph-specific terminology. Morph token is a occurrence of a morph in a text. Morph-types are the distinct morphs in the dataset. Frequency of a morph-type is
$$
f_{m}=\sum_{\forall m_i\in M,\ w_i:=m_i\in w_i}w_i
$$
for each occurrence of the morph sum the word frequency that the morph is part of.

\subsection{Type to token ratio (TTR)}
$$
\frac{\text{\# Morph types}}{\text{\# Morph tokens}}
$$

This metric is a abstraction of TTR to the morpheme case. Lower TTR means higher reusability of morphs. Compare two corpora with $10$ morph tokens:
\begin{itemize}
    \item only $5$ morph types leads to TTR being $\frac{1}{2}$
    \item only $10$ word types leads to TTR being $1$
\end{itemize}

\subsection{Morph-type entropy}
$$
H=-\sum_{w \in V} p(w)\log p(w)
$$
Where
\begin{itemize}
    \item $V$ is a set of all morph types,
    \item $p(w)$ is the probability of the morph-type $$\frac{ f_m } {\text{\# all morph tokens}}$$.
\end{itemize}

Higher entropy means that the frequencies are distributed more evenly across morph types. Lower entropy means that relatively small amount of morph types account for a large proportion of all tokens. In this sense, entropy captures the uncertainty of the morph distribution: if the next morph is difficult to predict because many morph types have comparable probabilities, the entropy is high.

\textbf{Perplexity} can be derived from entropy as
$$
\text{PP} = 2^H
$$
meaning the number of equally probable morph types.

\subsection{Hapax legomenon}
This metric is actually more reasonable in this context. It says how many morphs are unique for only a single word-type. It again suggests the reusability of morphs in the language.

\subsection{Other statistics}

The basic statistical measures are listed here:
\begin{itemize}
    \item Average morpheme count per word-type
    \item Morph-type to word-type ratio = captures the repetition of morphs
\end{itemize}

\subsection{Metrics based on the type of morpheme}

\begin{itemize}
    \item Average root count in the word-type
    \item Deviation of suffix and prefix
\end{itemize}
